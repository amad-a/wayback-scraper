Kinetic paintings for Saleh

To Dr. Nabile Ali and to Saleh and designer

To download, please first click on the picture that you want, then wait while the full file behind it downloads. When the entire larger files appears, save it to a directory you have on your own computer. After you save it, click the back arrow on your browser to return to the main page and select again.

I have given you both a full color and a black and white of each so that you can select what looks better to you.

There are 5 sets. The first batch is done on my Kinetic painting program which I wrote for my own use in performance with musicians.. The next set hand drawn using the mouse as single images. The Third set is high-resolution photo images taken from the screen. The fourth set is a couple of images taken from kinetic paintings done in the mid 80's on the Amiga. And the last is a a series of self portraits done over a period of time on the web allowing people to view the development of the drawing. Please check elsewhere on my pages for further information.

If you wish to know a bit about my thoughts on painting read "An Aesthetic View." and/or the essay "Abstraction and Illusion." To learn about me read the "Short Biography" or the "Long Biography" or "The Resume."
