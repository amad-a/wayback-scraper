Mini Telephone Directory

  October 28, 1999: this section is not ready yet. Please check back in the next few days.
Public phones are widely available in Palestine especially in towns and cities. Some are coin-operated and most use calling cards. The country code is 970 but 972 is still in use. If you are calling from overseas, drop the leading 0. Phone services are provided by PalTel (Nablus, Tel. 09-237-6225). Cell phones are also very common.
