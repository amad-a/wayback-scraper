GIF Builder
GifBuilder is a scriptable utility for creating animated GIF files. You can input an existing animated GIF, a collection of PICT, GIF, TIFF, and/or Photoshop files, a QuickTime movie, a PICS file, an Adobe Premiere FilmStrip 1.0 file, or the layers of an RGB or grayscale Adobe Photoshop 3.0 file. Output is a GIF89a file with multiple images. Version 0.5 features frame icons in the Frames window, filters, transitions, animation cropping, and more.
