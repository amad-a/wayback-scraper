Backing Up Your website

One of the most annoying things in the web design and the web administration is the web failure or loss issue, which results a lot of work in order to replace the lost data or redesign the hole page all over again. One way to avoid such things is the backup process which simplifies the restoring of lost files, and saves a lot of time and money.

Backup should be make daily on the webpages that had newly webpages add every day, The next steps describes how to back up your site in just a few minutes :

user your Favorite FTP program to log on your system .

Make a new directory in your local drive and copy all the files from your system to the Local Drive (c: for example).

If you don't have the WINZIP program, then it Time To install It .

 

Click On the New Icon on the Top of the Screen


.

 

In the file name field write your backup file name and Click Ok

 

 

Select all the requested files for backup and make sure that the Include subfolders icon is checked

Replay By Yes to the Popup Question

 

That's It Congratulation You have now a zip file that contains all your webpages .
