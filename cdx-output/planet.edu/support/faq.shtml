A4)- E-mail can be considered as a mail box where your messages that are send through the net can be stored. The E-mail Format is : Yourname@YourISP.Extention
where the name is Your username and the ISP is the name of the company that provides you with the Internet service and the extension is the type of the company.

Example:- webmaster@planet.edu

A5)- A home page is a place where you can put information about yourself on the web, the information may vary from simple one to more complex and advance ones, a page may contain text information, images, media,videos and other multimedia options. The home page must be hosted by an ISP so as to provide a 24H accessible page.

 

A6)- To Change the default Home page Go to Edit in the Netscape browser then Preferences --> Navigator, and Change the Right Side Location to the Home page You Would Like the Browser to open With, Then Click OK.

A7)- The home page design is one an interesting process and can be done by the owners or by the ISP hosting that Site, the Home page cost vary depending on the size type and techniques used in designing that page.

A8)- The main designing is HTML(Hyper Text Markup Language),also the page can be implemented using JavaScript or even Java applets, on the other hand the is a large verity of plugins used in pages to give the page a professional look such as shockwave, Active-X and SMIL . Click Here for a Tutorial

A20)- TCP/IP stands for Transmission Control Protocol/Internet Protocol.

A21)- Protocol is the language that the computers use to communicate with each other, it is a set of commands an instructions that used between computer so as they can interact with each other.

A22)- CGI stands for Common Gateway interface, which is a way that the user can interact with the server he is working on, Most Common examples for a CGI applications is the Submit Forms that Collects information's from the user and send them back to the server that respond to the sender.
