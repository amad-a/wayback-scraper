Java Programming Language

Java Centre- http://www.java.co.uk
Interested in Java? If so, you may find it worthwhile to visit Java Centre. Based in England, this site includes news, information, and demonstration applets. This site is aesthetically pleasing, but some of the graphics and frames may take a few moments to load. Whether you program in Java or you just enjoy checking out clever new Java applets, the Java Centre is an entertaining and educational resource.

Java FAQ Archive- http://www-net.com/java/faq
If you, as either a user or a developer, have run into problems with Java and need to know about any aspect of it, stop by this Web site. It's a central source for Java FAQs on the Net. At this index you can spend valuable time reading local documents— the General Java FAQ and Windows 95 Java FAQs, in both English and French. You can also find links to plenty of other Java resources, including documentation on OS/2 Java, Sun's WorkShop, Java security, Java on Linux, and Java networking. Other good resources that are listed here include questions and answers from the comp.lang.java.* newsgroups, the Java Newbie FAQ, the Object-Orientation FAQ, and Java tutorials. Check the TeamJava list of links for even more information.

Java Programming Resources- http://www.apl.jhu..edu/~hall/java
This Web site provides a huge one-stop source of Java-related links and other Web development information. Conveniently gathered on one page, this sizable index includes lists of links, along with helpful descriptions, to a variety of FAQs, tutorials, and documentation, with the official API and language specifications. You'll find links to lists of Java books and overviews of specific Java topics such as object-oriented programming, graphical user interface elements, threads, and network programming. If you want to take advantage of available software, there are links here to download sites for the Java Developer's Kit and Java-enabled browsers. Everything is neatly categorized in the table of contents at the top, so you should have no trouble finding what you need quickly.

Java Programming for the Internet- http://java.sun.com
Everyone involved with or interested in Hot Java should visit Sun Microsystems's original Hot Java home page. This site is your main source for getting the latest official Java news, applet demos, and licensing information. Check the Developer's Corner for the Java Developer's Kit and a comprehensive list of programming documentation. You can also download the latest versions of Hot Java browsers and ports. Browsing this site is a breeze; the layout is very organized and labeled with clever Java icons. The Hot Java Home Page is very popular among Internet developers; if you have trouble accessing this particular URL, you can view the same contents at a number of mirror sites.

Java World- http://www.javaworld.com
Introduced by Sun Microsystems in 1995, the Java scripting language was hailed as one of the most promising Web tools since HTML. Today the language has its own trade magazine, JavaWorld— a slick site for programmers and the Java-curious. Hands-on columns discuss the technical aspects of scripting Java applets, while the News & Views section has suggestions for using applications in the real world. You don't have to be a tech-head to appreciate Java. If you find the complex scripting process a little out of your league, try the Resources page, which has links to prime Java-enhanced sites. Note that some browsers may not be Java-compatible. Check this site for information on software supporting Java applets.
