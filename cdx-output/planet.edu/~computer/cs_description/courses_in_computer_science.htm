Courses Description                                                     

CS101 Fundamental Computing Skills

     An introduction to the use of the computer for those students in areas where the computer will be a useful, but not necessarily a vital, professional tool. Covers general problem solving, development of algorithms, computer hardware and software, data handling, spreadsheets, graphics, programming in the PASCAL, and applications of the computer in technology.

A class designed for those with no prior computer exposure or those students needing to build their mathematical skills and are taking an appropriate math class before starting the beginning computer sequence. Covers hands-on use of a computer, general computer knowledge, introduction of a programming language, problem solving, algorithms, practice in the programming process, overview of the computer science profession.

CS102 Fundamentals of Programming:

   General introduction to computers, hardware and software. Problem solving. Top- down algorithmic development. Data types: scalar and structured. Program development using a programming language. Introduction to the use of some application packages.

      A course in computing for students in disciplines where the computer is a vital professional tool. Focuses mainly on the FORTRAN language (coding source program, unformatted I/O, integer and real arithmetic, control statements, intrinsic functions, formatted I/O, arrays, implied loops subprograms, common statement and applications indifferent fields). Attention also given to program design and style, algorithm design, problem solving, numerical errors and their prevention, hardware and software.

        

CS115 Scientific Programming with FORTRAN:

      An introduction to the use of the computer for those students in areas where the computer will be a useful, but not necessarily a vital, professional tool. Covers general problem solving, development of algorithms, computer hardware and software, data handling, spreadsheets, graphics, programming in the BASIC language, numerical errors and their prevention, and applications of the computer in technology. This course does not apply toward a Computer Science, Computer Information Systems, or Management Information Systems concentration, major or minor. Credit may not be earned in this course and INSY118.

         

CS211 Data Structures & Algorithms:

   Basic concepts of information as data. Lists and strings. Tree structures. Storage devices and structures. Sorting and searching techniques. Storage allocation and collection.

     An introduction to programming methodology using the C programming language including computer usage within a UNIX environment, problem solving, algorithm development, control structures, arrays, program style, program design, code documentation techniques, and program correctness as well as a brief overview of computer systems and computer history. Three lectures and one laboratory each week.

      A continuation of the study of programming methodology by examining program specifications, design, coding, correctness, and style with additional coverage of pointers and arrays, and an in-study of data structures and recursion. Includes the study and implementation of simple lists, stacks, queues, and files as well as a brief overview of computer ethic and the impact of computers upon society.

      A continuation of the study of programming methodology with particular emphasis on program specification, design, and implementation of data structures and algorithms such as manipulation, searching, sorting, and traversal for linked structures, trees, and hash tables.

           

CS212 Data Structures & Algorithms:

 Continuation of Data Structures and Algorithms I. Software design, graphs and algorithms, which operate on them. Fundamental algorithms for searching, sorting, string matching, graph problems, and their analysis.

                                                  

CS311 Object Oriented programming:

    A survey of the structure and design of modern programming languages. Topics include procedure, control, object and data abstraction, modularization and concurrency, as well as vector operations, list processing, string manipulation, logical inferencing and exception handling.

                                                 

CS315 Advanced Programming Techniques:

          The state of art in computer languages and in programming techniques.
