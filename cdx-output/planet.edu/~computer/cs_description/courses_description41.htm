Courses Description  

CS441 Advanced Database Systems:

    Database design and theory, concurrency, distributed databases, integrity, security, query optimization. Economic tradeoffs involved in using database packages. Survey of available database packages. Class work to include a term project and readings from the literature.

          

CS444 Introduction To Software Engineering:

    Provides an in-depth study and practice of the system's engineering methodologies as applied to the complete life cycle of software systems. Design and implementation of large software systems. Includes systems analysis, design and requirements; data collection, analysis, organization and documentation; feasibility analysis; human-machine interface analysis; logical design specifications; input/output design; data file and database design; program design and processing control design; implementation; testing; and maintenance and security issues. Continuation of the study of the system life-cycle and group software development projects. Includes implementation, testing, maintenance, and security of a major software project. System theory, system development life cycle, initial investigation, feasibility study, analysis, design,implementation and post implementation. Information gathering,system flowchart, Gantt charts, tools of structured analysis,cost/benefit analysis. Stages of system design. A case study.

           

CS448 Management Information systems:

    Management information, structure of an MISS, organizing the data processing activity, office automation, decision making process,information systems plan, and project management. Computer-Center Management Covers specialized concepts required for computer-center management including personnel, documentation, standards, inventory, estimating, budget preparation, project management, site audits, security, casualty recovery and backup, typical growth patterns, applicability of database management systems, and maintenance strategies. Particular attention is given to the needs analysis, technical and financial aspects of computer acquisition, site preparation, and facilities planning.

           

CS352   Formal Languages & Syntactic Analysis:

     A survey of the fundamental mechanisms which underlie computer science, including finite state machines, tape machines, Turing machines, push-down automata. Corresponding formal language constructs and computability criteria will be covered.

         

CS355 Automata Theory:

       An introductory overview of the theory and utility of finite state machines and Turing machines in computer science. Deterministic and non deterministic finite automata, multitape Turing machines, universal Turing machine.

           

CS455 Compiler Design:

       Formal language and automata. Overview of compiler phases. Context free grammars. Syntax and directed translations. Techniques used inlexical scanning, parsing and symbol table implementation. Error diagnostic recovery.

                                                

CS361 Logic programming:

       History and theoretical foundations of automated theorem proving and logic programming. Includes a survey of various logic programming paradigms.
