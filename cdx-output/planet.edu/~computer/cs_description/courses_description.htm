Courses Description                                                     

CS316 Programming Languages:

    A survey of the structure and design of modern programming languages. Topics include procedure, control, object and data abstraction, modularization and concurrency, as well as vector operations, list processing, string manipulation, logical inferencing and exception handling. logic programming. Includes a survey of various logic programming paradigms.

                                                 

CS411 Design and Analysis of Computer Algorithms:

    Techniques used in analysis of algorithms. Design and evaluation of algorithms related to graphs, trees, prime numbers, memory management and text processing. Computer models, design of algorithms, sort/order problems.Integer Dan polynomial arithmetic, pattern recognition, algorithms on graphs. Matrix operation, the fast Fourier transform, NP-complete problems, intractable problems.

                                           

CS221 Discrete Structures I:

        Set theory, prepositional logic, Boolean algebra, relations and functions, counting theory and combinatorics, graph theory,probability and statistics, introduction to formal language theory.

                                                  

CS222 Discrete Structures II:

     Survey of discrete deterministic and non-deterministic systems useful to computer scientists, including number systems, modular arithmetic, graph theory, elements of probability, reliability, queuing systems, and networks.

                                        

CS323 Discrete System Simulation:

      Simulation and queue in models. Random numbers generation. Statistical sampling and analysis of data. Simulation languages and selected applications.

      Techniques in simulation on a digital computer. Generation of random numbers, Monte Carlo techniques, use of simulation languages. Development of simulation models for specific problems.

                                                  

CS326 Performance Evaluation of Computer Systems:

        Introduction of Computer System performance modeling and simulation using SIMSCRIPT. Topics covered include memory, processor and input/output subsystem model of computer systems.

       A study of the systematic applications of the scientific method to computer system performance. Problems directly addressed are performance projection, configuration design, system tuning, upgrading, and planning.

                                                 

CS324 Operations Research:

        Linear programming, properties of a solution to linear programming problems, simplex, duality, dual simplex method, sensitivity analysis, applications, transportation, assignment problems or network flow problems.
