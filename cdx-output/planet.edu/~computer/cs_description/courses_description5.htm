Courses Description  
CS363 Artificial Intelligence:

       An examination of the artificial intelligence approach to modeling cognitive processes. Includes  an introduction to heuristic methods, problem representation and search methods, classic AI techniques, and a review of the controversial issues of the AI paradigm of cognition as computation.

           

CS456 knowledge Base Systems:

     Topics include Knowledge, Knowledge representation, formal logic, database programming, and recursive programming. selected language features including rules, arithmetic, type predicates, meta-logic predicates, cuts, and negation are introduced with programming examples.

     Theory and practice of knowledge-based system const. Topics will include knowledge-base construction, inference engines, reasoning from incomplete or uncertain information, and user interfaces.

           

CS271 Introduction to Unix and C Programming:

       An introduction to the UNIX operating system, including basic system commands, editing, shell programming, and use of software management tools. An introduction to the C programming language.

                                            

CS371 UNIX Systems Administration:

     Management and administration of UNIX computer systems. Includes account, process, disk, terminal, and network management; installation, startup and shutdown; backups and troubleshooting. Hands-on lab emphasized.

       

CS471 Computer Operating Systems:

     Operating systems history. Basic issues in concurrency, dead lock control, synchronization, scheduling, memory management, process management, resource management, and protection and access control.Implementation of parts or a small operating system.

      Introduction to the theory and structure of operating systems. Issues of efficient resource allocation and management in multi-programming systems.

    Continuation of Operating Systems I with emphasis on comparing different systems. There is a major project for this class that deals with using a main-frame with no operating system. Offered alternate years only.

 

           

CS472 Computer Operating Systems Design:

       A continuation of CS471. Advanced topics in operating systems comparative studies of different types of operating systems.

      Advanced topics and current research in operating systems. Topics may include system structures and algorithms, reliability, security, distributed systems, system modeling, as well as recently published research in these and other areas. Study of commercially available and experimental operating systems highlighting these concepts. Class work to include a term project and readings from the literature.

     Continuation of Operating Systems  with emphasis on comparing different systems. There is a major project for this class that deals with using a main-frame computer with no operating system. Offered alternate years only.

                                     

CS474 Real-Time Computer Applications:

     Programming applications in real-time environment. Attention is given to the design of applications programs in a multitasking environment. Examples are considered from such areas as process control, robotics, signal analysis, networking, and multiwindow workstation software.
