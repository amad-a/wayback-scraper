This self paced self study guide on Operating systems includesall the information you need to learn about Windows95 and otherOperating systems quickly and effectively. The on-line notes are free, but you can purchase the CD-ROM with many other of our on-linemodules, and is packed with heaps of examples, inter-active tests, and a complete set of notes.

This self study guide introduces you to Operating systems,specifically, MS-DOS, Unix and Windows95. It comprises thefollowing sections

  • operating systems
  • computer architecture and terminology
  • revision of MSDOS
  • an overview of graphical window systems
  • introduction to UNIX
  • introduction to Windows95

Course Objectives
By the end of this module, you should be able to

  • understand basic computer terminology
  • use Windows95 to accomplish a number of tasks
  • manage folders and files using Windows95
  • configure the desktop
  • print documents
  • copy and share information between applications and files

Work through each of the sections at your own pace. Try tospend an hour a day reading the material, and doing any exercisesinvolved. The entire course takes about 40 hours of work. Forsome sections, you will need access to a UNIX computer. Forothers, access to a Windows95 computer is required.
