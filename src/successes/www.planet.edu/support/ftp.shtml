FTP Your Web ...

To put your Web on line you have to have the following :

  1. Web Account (username + password)
  2. FTP program such as CuteFTP or WS_FTP

We will use the following assumptions:

  • The web address is http://www.planet.edu/~myaccount (my account is your username)
  • The password is mypassword
  • The FTP program is CuteFTP

Step 1 : -

- Start -->Programs --> CuteFTP -->CuteFTP

- From the POP-UP Screen Click on Add Site

 

- Fill in the blanks as bellow

- Click OK

You will be back to the previous page with the Site Name shown in the right white Space

- Double Click on the Site Label

 

Step 2 : -

The following screen has mainly 2 useable parts:

  • The Server View (right)
  • The Local PC View (left)

The Local PC is our PC hard disk in which you store your web page and you can change to the directory by using the up arrow located on the top of this view

The Server View is the one in which your web must be hold on, by default their will be Two Directories

  • Public_html
  • cgi-bin

Public_html

Is the Directory the holds all the HTML files , The first file must be named index.html other wise the page will not be displayed.

You can place as much directories as you need in this directory

cgi-bin

Is the place where your .cgi or .pl scripts are placed in, this directory is useful if you have a guestbook or an interactive forms

- Double click on Public_html

- Mark all the files you need to transfer from your local View and drag them to the Server side.

- Check you page .

ps: the next time you run your ftp program just double click on the site name and you will be in the server.
