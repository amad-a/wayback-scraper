General information

Your account comes with several free ASP cgi scripts. They are already installed and ready for use. To change the configuration of a script, edit configuration files in the directory of this script. To begin using the script, click the icon next to the script location.

*All the scripts are freeware and reference links to the script developers are included.

Name Description Location
ConquerChat ConquerChat allows your users to conduct virtual meetings on-line. /cp/scripts/ASP/chat/ 
Web Wiz Graphical Counter Web Wiz Graphical Counter allows you to display how many visitors your website has received. /cp/scripts/ASP/counter/ 
WebGuestbook WebGuestBook is a self-sustainable HTML-based child-safe application providing an opportunity to your site's visitors to leave their remarks and impressions. /cp/scripts/ASP/guestbook/
