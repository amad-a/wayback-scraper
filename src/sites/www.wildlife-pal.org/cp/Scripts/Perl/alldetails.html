General information
Your account comes with several free Perl cgi scripts. They are already installed and ready for use. To change the configuration of a script, edit configuration files in the directory of this script. To begin using the script, click the view icon next to the script location.

*All the scripts are freeware and reference links to the script developers are included.

Name Description Location
EveryChat EveryChat allows your users to conduct virtual meetings on-line. /cp/scripts/Perl/Chat/ 
E-Guest E-Guest is a full features guestbook system. /cp/scripts/Perl/Guestbook/ 
Basic Graphical Counter Basic Graphical Counter is a simple script to display a counter on your Web page. It should be include some tag IMG of your Web page.

/cp/scripts/Perl/counter/
