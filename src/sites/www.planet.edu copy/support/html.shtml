Creating Your Own Web Page


The first Step is to Run You HTML Editor " In our Case it's the Netscape Composer"

- Double Click on Netscape icon in your desktop.

- From the Bottom Left side of the Netscape Browser Screen click on the last icon "Composer"

- You will have the Below Screen

- In the White area Write Down The Text You Need

- To Center The Text, Mark the Text with the Mouse and click on the align icon on the right side of the screen.

- To Enlarge The Text Choose a Size From The List

- To Give a Color For the Text Click in The Color box and Choose a Color

- To Save The HTML file choose File from the Menu then Save As

- In the file name write the name you want . "The first file must be named index.html"

- You can give your HTML document more Professional look by adding images and lines.



New: For a new HTML Document
Open: To open a previous Document
Publish: to put your page on the net
Preview: to view your document in the Netscape
Cut: to cut a selected Text
Copy: to Copy a Selected Text
Past: to insert Text in a defined place
Print: To print the HTML
Find: To find a Text in the Document
Link: To Link a selected document to a place
Image: to add an image to the HTML document
H.Line: to Add a horizontal Line
Table: to Add a table to the Document
Spelling: To check your Document Spelling
