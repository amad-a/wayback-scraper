Planet-Support-Network Configuration-

  • Click on START, then SETTINGS, then CONTROL PANEL.



  • Double-click on NETWORK.

  • Make sure that you are on the Configuration tab folder and look in the first text window entitled 'The following network components are installed:' to insure that you have a Dial-up Adapter and a TCP/IP Protocol installed. otherwise follow the instructions following to add these in.



  • To add in a Dial-up Adapter click ADD (skip if you already have one) and select ADAPTER from the list of four choices and then ADD once again.



  • On the left, scroll down until you can select MICROSOFT and then select DIAL-UP ADAPTER from the two choices on the right.



  • Click OK, which will load in the dial-up adapter and take you back to the main Network window.

  • Remove every thing except the Adapter you had just added.

  • To add in the TCP/IP protocol, select ADD and then select PROTOCOL from the four choices and then ADD to move on to the selection types.



  • For the manufacturer, select Microsoft and then TCP/IP as the protocol.



  • Click OK, which will take you back to the main Network window. (If it asks to specify the Adapter, do so and select Microsoft and Dial-up Adapter, then click OK)


  • Windows 95 will add TCP/IP. Do not close the Network window at this point
