Planet-Support-Dialup configuration

 

  • Click on START, PROGRAMS, ACCESSORIES, then DIAL-UP NETWORKING.
  • Double click on MAKE NEW CONNECTION. (If this is the first time you are using dial-up networking it will automatically start the start-up wizard, which will activate the make new connection.) You can name the icon "Planet" when prompted for the name of the computer you are dialing. Make sure that your modem type is selected, if not select it and click NEXT .
  • Enter in the correct phone number as listed below and click NEXT then FINISH on the next screen that follows.
    Area Code: 02
    Telphone : 5841611
    Country Code: Israel
  • You will now have a planet Network icon in your Dial-up Networking folder. You can click on this icon with your left mouse button, while holding the button down, and drag the icon out into your desktop. This will create a shortcut for quicker access to login.
  • With the right button on your mouse, click on the newly created Planet Icon and then select PROPERTIES from the pop up list.
  • Select SERVER TYPES, which will either be the second tab folder or a button at the bottom of your window.
  • The Type of Dial-up Server should already be set as PPP: Windows 95, Windows NT 3.5, Internet, if it is not set as this, please set it to this now.
  • On Advanced Options you want only Enable Software Compression selected. If Log on to Network is selected, un-select it by clicking the checkbox.
  • On Allowed Network Protocols you will want only TCP/IP selected. If IPX/SPX Compatible and NetBEUI are selected un-select them by clicking on the checkbox.
  • Click OK all the way back to the main Dial-up Networking Window with the Planet Icon.
  • Double-click on your Planet icon you created in the Dial-up networking.
  • In User Name put in your login name in all lower case (i.e. Webmaster), and your password (it's case sensitive so make sure you have it in all lower case unless it was specifically set up differently.)
  • Hit the CONNECT button at the bottom. Windows will tell you when you are connected or zoom your icon down into your toolbar, by your clock. (after two minutes or so.)
