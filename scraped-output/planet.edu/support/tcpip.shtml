Planet-Support-TCP/IP Configuration

 

  • In the Network window select TCP/IP, click PROPERTIES. This will bring up a new Window with six tab folders.
  • Select the tab folder marked DNS CONFIGURATION and fill in with the following information.
    Host name: Your username (login)
    Domain name:planet.edu
  • The Following Are Optional
    DNS server:192.116.10.1 (enter and click ADD ),Domain suffix: planet.edu (click add) .
  • Select the GATEWAY tab folder and enter the number 192.116.10.2 in New Gateways and then click ADD to install it.
  • Select the WINS CONFIGURATION tab folder and select Disable WINS Resolution.
  • Click OK to dismiss the TCP/IP Properties window and OK once again to dismiss the Network window.
  • You will get a prompt asking if you want to restart you computer, do so by selecting YES, which will then make these configuration settings active.
