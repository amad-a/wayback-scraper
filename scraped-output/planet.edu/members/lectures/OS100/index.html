This self paced self study guide on Operating systems includes all the information you need to learn about Windows95 and other Operating systems quickly and effectively. The on-line notes are free, but you can purchase the CD-ROM with many other of our on-line modules, and is packed with heaps of examples, inter-active tests, and a complete set of notes.

This self study guide introduces you to Operating systems, specifically, MS-DOS, Unix and Windows95. It comprises the following sections

  • operating systems
  • computer architecture and terminology
  • revision of MSDOS
  • an overview of graphical window systems
  • introduction to UNIX
  • introduction to Windows95

Course Objectives
By the end of this module, you should be able to

  • understand basic computer terminology
  • use Windows95 to accomplish a number of tasks
  • manage folders and files using Windows95
  • configure the desktop
  • print documents
  • copy and share information between applications and files

Work through each of the sections at your own pace. Try to spend an hour a day reading the material, and doing any exercises involved. The entire course takes about 40 hours of work. For some sections, you will need access to a UNIX computer. For others, access to a Windows95 computer is required.
